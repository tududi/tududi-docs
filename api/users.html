Users API

User management and profile endpoints.

Get Current User​

Get the authenticated user's profile.

GET /api/users/me

Response​

{
  "success": true,
  "data": {
    "id": 1,
    "username": "user@example.com",
    "name": "John Doe",
    "email": "user@example.com",
    "preferences": {
      "language": "en",
      "timezone": "America/New_York",
      "theme": "light"
    },
    "createdAt": "2025-01-01T00:00:00Z",
    "updatedAt": "2025-10-25T10:00:00Z"
  }
}

Update Current User​

Update the authenticated user's profile.

PUT /api/users/me
Content-Type: application/json

{
  "name": "John Smith",
  "preferences": {
    "language": "es",
    "theme": "dark"
  }
}

Response​

{
  "success": true,
  "data": {
    "id": 1,
    "name": "John Smith",
    "preferences": {
      "language": "es",
      "timezone": "America/New_York",
      "theme": "dark"
    },
    "updatedAt": "2025-10-25T15:00:00Z"
  }
}

Change Password​

Change the current user's password.

POST /api/users/me/password
Content-Type: application/json

{
  "currentPassword": "OldPassword123!",
  "newPassword": "NewPassword456!"
}

Response​

{
  "success": true,
  "message": "Password updated successfully"
}

Get User Statistics​

Get task statistics for the current user.

GET /api/users/me/stats

Response​

{
  "success": true,
  "data": {
    "totalTasks": 245,
    "completedTasks": 189,
    "incompleteTasks": 56,
    "overdueTasks": 8,
    "tasksThisWeek": 23,
    "completionRate": 0.77,
    "averageTimeToComplete": 172800000,
    "productivityTrend": "increasing"
  }
}

User Preferences​

Get Preferences​

GET /api/users/me/preferences

Update Preferences​

PUT /api/users/me/preferences
Content-Type: application/json

{
  "language": "fr",
  "timezone": "Europe/Paris",
  "theme": "dark",
  "dateFormat": "DD/MM/YYYY",
  "timeFormat": "24h",
  "notifications": {
    "email": true,
    "telegram": false,
    "desktop": true
  }
}

Avatar​

Upload Avatar​

POST /api/users/me/avatar
Content-Type: multipart/form-data

avatar: [image file]

Delete Avatar​

DELETE /api/users/me/avatar

Connected Apps​

List Connected Apps​

GET /api/users/me/connected-apps

Revoke App Access​

DELETE /api/users/me/connected-apps/:appId

Export Data​

Request a data export (GDPR compliance).

POST /api/users/me/export
Content-Type: application/json

{
  "format": "json"
}

Response​

{
  "success": true,
  "message": "Export requested. You will receive an email when it's ready.",
  "exportId": "exp_123456"
}

Delete Account​

Permanently delete the user account.

DELETE /api/users/me
Content-Type: application/json

{
  "password": "MyPassword123!",
  "confirmation": "DELETE"
}

Admin Endpoints​

Note: Requires admin privileges

List All Users​

GET /api/admin/users

Get User by ID​

GET /api/admin/users/:id

Update User​

PUT /api/admin/users/:id

Delete User​

DELETE /api/admin/users/:id

Ban User​

POST /api/admin/users/:id/ban

Unban User​

POST /api/admin/users/:id/unban

Error Responses​

Invalid Password (400)​

{
  "success": false,
  "error": {
    "code": "INVALID_PASSWORD",
    "message": "Current password is incorrect"
  }
}

Weak Password (400)​

{
  "success": false,
  "error": {
    "code": "WEAK_PASSWORD",
    "message": "Password does not meet requirements"
  }
}

Examples​

Update User Profile​

const response = await fetch('http://localhost:3000/api/users/me', {
  method: 'PUT',
  headers: {
    'Content-Type': 'application/json'
  },
  credentials: 'include',
  body: JSON.stringify({
    name: 'Jane Doe',
    preferences: {
      language: 'es',
      theme: 'dark',
      timezone: 'Europe/Madrid'
    }
  })
});

const data = await response.json();
console.log('Profile updated:', data);

Change Password​

const response = await fetch('http://localhost:3000/api/users/me/password', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  credentials: 'include',
  body: JSON.stringify({
    currentPassword: 'OldPassword123!',
    newPassword: 'NewPassword456!'
  })
});

if (response.ok) {
  console.log('Password changed successfully');
}
